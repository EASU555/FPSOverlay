# FPSOverlay

Windows 游戏性能监控悬浮窗。它在游戏中显示 FPS、平均 FPS、帧时间、CPU/GPU 占用与温度、功耗、显存和系统内存等指标，并提供设置中心和托盘控制。

## 项目用途

- 以横向游戏监控条显示实时性能数据。
- 识别游戏进程，在“仅桌面显示”模式下避免普通应用的全屏或最大化状态被误判为游戏。
- 通过本地硬件传感器、NVIDIA 数据源和可选 AIDA64 共享内存读取指标。
- 提供功耗记录、低 FPS 提醒、网络速率、开机自启、显示布局与诊断信息等本地功能。

## 使用说明

请先阅读 [USAGE.md](USAGE.md)。该文档包含运行前提、启动步骤、游戏显示行为、设置入口和常见排查方法。

## 构建

这是 Windows 原生 C++ 项目，解决方案文件为 `FPSOverlay.sln`，项目文件为 `FPSOverlay.vcxproj`。请在安装 Visual Studio 的 C++ 桌面开发组件后，以 Release 配置生成；仓库中的 `build-msvc.bat` 和 `build-ui-qa.bat` 分别用于正式构建和 UI QA 构建。

## 说明

- 此仓库仅包含源代码与必要资源，不包含已编译 EXE、PDB、日志或本机配置。
- 硬件数据会因显卡、驱动、传感器可用性和权限不同而变化；不可用的项目会显示为 N/A 或使用既有回退来源。
- 本项目不包含游戏注入、驱动或网络上传功能。
