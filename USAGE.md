# FPSOverlay 使用说明

## 运行前提

1. 使用 Release 构建生成程序，或使用项目发布目录中的正式程序。
2. 首次运行后通过托盘图标打开设置中心。
3. 如需更完整的硬件指标，请确认对应显卡驱动和本地硬件监控来源可用；程序会保留缺失指标的 N/A 状态。

## 基本操作

1. 启动程序，屏幕上会出现横向性能悬浮窗，托盘区会出现程序图标。
2. 右键托盘图标可显示或隐藏 Overlay、打开设置、重新扫描传感器或退出程序。
3. 在设置中心调整显示指标、刷新间隔、布局、低 FPS 提醒、网络速率、功耗显示和开机自启等选项。
4. 使用默认 `F9` 可循环切换可用的 Overlay 布局；可在设置中修改热键。

## 游戏与桌面显示

- 默认按游戏优先逻辑选择 FPS 目标，Overlay 在已识别游戏中保持显示。
- 启用“仅桌面显示”后，普通应用即使最大化或全屏也不会被作为游戏监控目标。
- 如果某个游戏只能采集 FPS 而看不到 Overlay，可能受游戏或反作弊的窗口层级限制；本程序不采用进程注入方案。

## 排查

- 指标显示 N/A：从托盘菜单执行“重新扫描传感器”，再检查显卡驱动和传感器来源。
- 游戏未被识别：在设置和日志中检查目标 PID、FPS 来源与游戏识别诊断。
- 休眠唤醒后异常：等待程序恢复硬件轮询；若托盘图标未恢复，可退出后重新启动。

日志和诊断文件由程序保存在发布目录附近的本地位置，仅用于排查。
